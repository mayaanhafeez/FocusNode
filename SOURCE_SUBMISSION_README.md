# FocusNode — source code submission

This archive contains the complete, unbuilt source for the FocusNode browser
extension, submitted per Mozilla's requirement for add-ons that use a build
step (this one uses Vite + `@vitejs/plugin-react` to compile and bundle the
popup/options UI).

## What's generated vs. hand-written

- `focus-react/` — hand-written React source. Building it with Vite produces
  the bundled, minified files that ship inside
  `FocusModeWebsiteBlocker/dist/` (e.g. `popup-*.js`, `options-*.js`,
  `storage-*.js`). Those generated files are **not** included in this
  archive — follow the steps below to reproduce them.
- `FocusModeWebsiteBlocker/manifest.json`, `sw.js`, `blocked.js`,
  `blocked.html` — hand-written, not generated by any build step.

## Build environment used

- Node.js v26.5.0
- npm 11.17.0

## Steps to reproduce the submitted build

```sh
cd focus-react
npm install
npm run build
```

This runs `vite build`, which compiles the JSX/React source in
`focus-react/src/` and writes the bundled output directly into
`../FocusModeWebsiteBlocker/dist/`.

The complete, loadable extension is the `FocusModeWebsiteBlocker/` directory
after this build step: `manifest.json` + `sw.js` + `blocked.html` +
`blocked.js` + the freshly generated `dist/`.
